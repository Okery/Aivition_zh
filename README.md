Aivition
=
Aivition 是一款综合性的图片处理工具。它的软件体积小，启动速度快，可以用来看图，还可以瀑布流式浏览图片集。同时，它包含一张无限画布，就像在桌面一样可以自由排列组合图片；当需要处理图片时，只需右键图片，便会弹出编辑菜单，有常规的裁剪、旋转，还有抠图、高清放大、换脸等 AI 功能。

## :zap: 下载
[123云盘](https://www.123865.com/s/hDqXvd-CZU4H?pwd=Aivi#)
- 支持平台：Windows 10/11
- 便携版，无需安装，解压即用。
- AI 功能需要下载对应的权重文件
- 完全卸载此软件，需要先打开软件->设置->清除注册表，然后手动删除软件目录。已产生的图片编辑记录保存在图片所在目录中的 .aivition 文件夹内，可自行决定是否删除。
